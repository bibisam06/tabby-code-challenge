# 🐾 태비 팀 코딩 스터디 규칙

## 📌 스터디 규칙

1. **매주 최소 1회 코드 업로드하기**
2. **매주 금요일까지 Pull Request(PR) 올리기**
   - PR 올리면서 서로 코드 리뷰 진행하기
3. **각자 임베디드 파트 공부해오기**
4. **규칙 미준수 시 벌칙**
   - 금요일 저녁 간식/음식 사오기 (만원~2만원 사이)

## 📂 기타 안내

- 모든 코드는 공동 GitHub Repository에 업로드합니다.
- PR에는 간단한 작업 내용이나 배운 점을 작성해주세요.
- 코드 리뷰는 건설적으로! 서로 성장하는 데 초점을 맞춥니다.
